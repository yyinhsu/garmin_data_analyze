# Project Instructions

## Database Schema

資料庫架構請看 `DATABASE_SCHEMA_ZH.md`（中文）或 `DATABASE_SCHEMA.md`（英文）。
若架構有變動，請同時更新兩份文件。

## Python Environment

Always use `./.venv` as the Python environment for this project.

```bash
.venv/bin/python script.py
# or
.venv/bin/python -c "..."
```

Do NOT use the system `python3`.
